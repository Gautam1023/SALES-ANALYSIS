# Sales Dashboard - Power BI

## Project Overview

This project presents a Sales Performance Dashboard developed using Microsoft Power BI. The dashboard provides insights into sales trends, profit analysis, regional performance, customer segmentation, and product category analysis.

## Features

* Total Sales Analysis
* Profit and Revenue Tracking
* Region-wise Performance
* Category-wise Sales Breakdown
* Monthly Sales Trends
* Interactive Filters and Slicers

## Tools Used

* Microsoft Power BI
* Excel/CSV Dataset
* DAX
* Power Query
